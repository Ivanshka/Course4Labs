ЛАБА 7:
1. Где хранится база данных вашего приложения? Как создать базу данных? Что
будет если не задать ей имя?
В операционной системе Android файл базы данных хранится в /data/data/<полное имя приложения>/databases/<имя базы>.db
Sqlitebrowser - помощник для создания БД.

2. Объясните назначение объектов SQLiteOpenHelper, SQLiteDatabase, Cursor.
класс SQLiteOpenHelper - с помощью которого можно создавать, открывать и обновлять базы данных.
Для управления базой данных SQLite существует класс SQLiteDatabase (методы query(), insert(), delete() и update()).
Объект Cursor, возвращаемый методом query(), обеспечивает доступ к набору записей результирующей выборки.

3. Как создать таблицу базы данных?
с помощью Sqlitebrowser.

4. Что такое класс – контракт и для чего он создается?
Класс контракта определяет константы, которые помогают приложениям работать с URI содержимого, именами столбцов, действиями намерений и другими функциями поставщика контента.

5. Какие методы нужно определить в классе-помощнике SQL Helper?
onCreate(), onUpgrade()

6. Зачем указывается номер версии базы данных? На что он влияет?
когда замечает, что номер версии базы стал другим, значит приложению пора обновиться.

7. Приведите пример выполнение команды SQL (например обновления таблицы).

8. Как вставить данные в таблицу? Что такое ContentValues?
Вставка - ContentValues.put(). ContentValues - объект для работы с данными.

9. Какой формат написания query? Какие дополнительные операторы можно
использовать в запросах?
query(таблица, столбцы, столбцы_для_условия_WHERE, значения_для_условия_WHERE...)

10. Как удалить данные?
Метод delete() класса SQLiteDatabas

11. В чем разница между методами getReadableDatabase() и getWritableDatabase()?
Метод getReadableDatabase создаёт или открывает базу данных. getWritableDatabase() - на запись.

12. Как прочитать данные из курсора?
с помощью запроса query.

13. Что такое CursorAdapter?
Адаптер, который предоставляет данные из Cursora ListView виджету.

14. Что должно выполняться в основном потоке, а что в фоновом?
основной поток - с которым взаимодействует пользователь и то, что отображается. Остальные - фон.

15. Для чего и как используют класс AsyncTask? Охарактеризуйте его методы:
назначение, выполнение, доступ к UI, тип параметра и порядок вызова.
Класс AsyncTask позволяет определить операции, которые будут выполняться в фоне.

16. Что означают типы в generic классе AsyncTask?
классы-оболочки (int - Integer).

17. Как запустить асинхронную задачу?
doInBackground()

18. Как отменить выполнение асинхронной задачи?
cancel()

19. Что такое ContentProvider?
Content Provider – это способ расшарить для общего пользования данные вашего приложения.
